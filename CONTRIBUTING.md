# Contributing

Thanks for contributing to Dental_v3.

## How to contribute

1. Fork the repository and create a focused branch for your change.
2. Keep changes small and clearly describe the problem being solved.
3. Update documentation when behavior or usage changes.
4. Run any available validation before opening a pull request.

## Pull requests

- Use clear titles and descriptions.
- Include relevant screenshots or examples when helpful.
- Keep pull requests limited to a single concern when possible.

# Code of Conduct

Please help keep this project welcoming and respectful.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Focus feedback on the work, not the person.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, intimidation, or disruptive conduct.
- Sharing private or sensitive information without permission.

## Reporting concerns

If you experience or witness unacceptable behavior, open a private security report or contact the repository owner directly.
